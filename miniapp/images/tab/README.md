# TabBar 图标说明

本目录存放 TabBar 图标占位文件。

## 当前状态
当前提供 SVG 格式的占位图标，用于开发预览。

## 注意事项
微信小程序 TabBar 仅支持 png / jpg / jpeg 格式图标。
在正式发布前，需要将 SVG 图标转换为 PNG 格式（建议尺寸 81x81px）。

## 需要的图标文件
- home.png — 首页图标（未选中，灰色 #999999）
- home-active.png — 首页图标（选中，深色 #333333）
- category.png — 分类图标（未选中，灰色 #999999）
- category-active.png — 分类图标（选中，深色 #333333）
- mine.png — 我的图标（未选中，灰色 #999999）
- mine-active.png — 我的图标（选中，深色 #333333）

## 图标规范
- 尺寸：81px x 81px
- 格式：PNG（建议带透明通道）
- 未选中色：#999999
- 选中色：#333333
