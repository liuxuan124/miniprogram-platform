## 〇、整体验收标准（Definition of Done）

1. 全站无假数据、无「功能开发中」死入口（搜索 `功能开发中 / mock / 写死常量` 清零）。
2. 核心链路端到端可复现：登录 → 装修发布 → 下单 → 支付 → 订单流转。
3. 安全红线项（S1/S2/S3）全部闭环，提供配置证据。
4. 列表三态（加载 / 空 / 失败重试）齐全；高频操作（导出 / 批量）补齐。
5. 生产日志与查询性能达标。
